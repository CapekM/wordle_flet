# What to Learn Next in Flet

## Navigation

- URL-based routing (`page.on_route_change`, deep linking)
- `NavigationBar`, `NavigationRail`, `NavigationDrawer`, `Tabs`, `AppBar`

## Forms & Input

- `TextField`, `Dropdown`, `Slider`, `Checkbox`, `Radio`, `DatePicker`, `TimePicker`
- Form validation patterns

## Rich Layout

- `Stack` (overlapping widgets), `ListView`, `GridView` (scrollable lists/grids)
- `ResponsiveRow` for adaptive layouts
- `ExpansionPanel`, `Card`, `DataTable`

## Drag & Drop / Gestures

- `Draggable`, `DragTarget`, `GestureDetector`

## Media

- `Image`, `Audio`, `Video` controls
- `FilePicker`, `Camera`

## Data Visualization

- `BarChart`, `LineChart`, `PieChart` (via `flet-charts`)
- `MatplotlibChart`, `PlotlyChart` integration

## Maps

- `Map` control with tile layers, markers, polygons (via `flet-map`)

## Canvas / Custom Drawing

- `Canvas` control for freeform drawing with paths, shapes, text

## Accessibility

- `Semantics` control for screen reader support

## Advanced Patterns

- `page.pubsub` for cross-session communication
- `page.session` for session-scoped state
- Explicit animations (`AnimatedSwitcher`, `AnimatedRotation`, etc.)
- Custom Flutter widget wrapping

## Deployment

- Building for iOS/Android with `flet build`
- Web deployment (WebAssembly via Pyodide, or server-side)
- Desktop packaging (Windows/macOS/Linux)
